# AWS DevOps Full Beginner Project

This repository contains the full DevOps beginner project built step-by-step with AWS, Linux, GitHub, Docker, and CI/CD.

## Structure
- aws/ – AWS resources, diagrams, CLI commands
- linux/ – Bash automation scripts
- github/ – GitHub workflows, CI/CD pipelines
- app/ – Application code
- diagrams/ – Architecture diagrams

## Goal
To master DevOps fundamentals and prepare for real-world cloud engineering.
